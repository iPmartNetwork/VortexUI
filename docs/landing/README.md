# VortexUI Landing Page

Single-page marketing site for VortexUI v1.3.1 (React + Vite + Tailwind).

**Author:** ali (zeus)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Produces a single-file `dist/index.html` (via `vite-plugin-singlefile`).
