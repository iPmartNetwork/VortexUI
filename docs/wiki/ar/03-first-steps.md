# الخطوات الأولى

!!! info "الوقت المقدّر"
    من تسجيل الدخول إلى رابط اشتراك عامل: **5 دقائق**.

---

## 1. تسجيل الدخول وتأمين حسابك

1. انتقل إلى عنوان لوحة التحكم (مثل `https://panel.example.com`)
2. سجّل الدخول ببيانات المسؤول التي أنشأتها أثناء التثبيت
3. **غيّر كلمة المرور** فوراً: الإعدادات → الملف الشخصي → تغيير كلمة المرور

### تفعيل المصادقة الثنائية TOTP

1. اذهب إلى **الإعدادات → المصادقة الثنائية**
2. انقر **تفعيل** — امسح رمز QR بتطبيق Google Authenticator أو أي تطبيق TOTP
3. أدخل الرمز المكوّن من 6 أرقام للتأكيد
4. احفظ رموز الاسترداد في مكان آمن

!!! warning
    بدون المصادقة الثنائية، أي شخص يملك كلمة مرورك لديه وصول كامل للوحة التحكم. فعّلها قبل تعريض اللوحة للإنترنت.

---

## 2. إضافة أول عقدة

=== "العقدة المحلية (خادم واحد)"

    إذا كانت لوحة التحكم ونواة البروكسي على نفس الجهاز:

    1. اذهب إلى **العقد → إضافة عقدة**
    2. اختر **عقدة محلية**
    3. اختر النواة: **Xray-core** أو **sing-box**
    4. انقر **إنشاء**
    5. تبدأ العقدة تلقائياً وتظهر "متصلة"

=== "العقدة البعيدة (معالج التسجيل)"

    لخادم منفصل:

    1. اذهب إلى **العقد → إضافة عقدة**
    2. اختر **عقدة بعيدة**
    3. يعرض المعالج أمر تثبيت بسطر واحد — انسخه
    4. ادخل عبر SSH إلى خادمك البعيد والصق الأمر
    5. انتظر حتى يتصل العميل — تظهر العقدة بحالة "متصلة"

!!! tip
    يتولى معالج التسجيل تبادل الشهادات، تثبيت النواة، وتسجيل الخدمة. لا حاجة لإدارة الشهادات يدوياً.

---

## 3. إنشاء أول اتصال وارد

إعداد سريع لـ VLESS + REALITY (موصى به لاتصالات مقاومة للرقابة):

1. اذهب إلى **الاتصالات الواردة → إضافة اتصال وارد**
2. اضبط الإعدادات:

    | الحقل | القيمة |
    |-------|--------|
    | البروتوكول | VLESS |
    | العقدة | عقدتك |
    | المنفذ | `443` |
    | النقل | TCP |
    | الأمان | REALITY |
    | الوجهة (SNI) | `www.google.com:443` (أو استخدم ماسح Reality) |
    | أسماء الخادم | `www.google.com` |
    | المعرّفات القصيرة | تُنشأ تلقائياً (أو أدخل خاصتك) |

3. انقر **إنشاء**

!!! tip "ماسح Reality"
    استخدم **الأمان → ماسح Reality** لإيجاد أفضل نطاقات SNI لموقع خادمك. اختر النتيجة الأعلى تقييماً لزمن استجابة منخفض واتصالات مستقرة.

---

## 4. إنشاء أول مستخدم

1. اذهب إلى **المستخدمون → مستخدم جديد**
2. املأ الحقول:

    | الحقل | القيمة |
    |-------|--------|
    | اسم المستخدم | `testuser` |
    | حدّ البيانات | `50 GB` (أو `0` لغير محدود) |
    | انتهاء الصلاحية | بعد 30 يوماً من الآن |
    | حدّ الأجهزة | `3` |
    | الاتصالات الواردة | اختر اتصال VLESS الوارد |

3. انقر **إنشاء**
4. يُنشأ توكن اشتراك المستخدم تلقائياً

---

## 5. اختبار رابط الاشتراك

1. من قائمة المستخدمين، انقر على اسم المستخدم لفتح صفحة التفاصيل
2. انسخ **رابط الاشتراك** (أو امسح رمز QR)
3. استورده في تطبيق العميل:

=== "v2rayNG (Android)"

    1. افتح v2rayNG → انقر **+** → **استيراد إعداد من الحافظة**
    2. أو امسح رمز QR مباشرة

=== "Clash Meta (سطح المكتب/الهاتف)"

    ```
    https://panel.example.com/sub/<token>?format=clash
    ```
    أضفه كرابط اشتراك في Clash.

=== "sing-box (iOS/Android)"

    ```
    https://panel.example.com/sub/<token>?format=singbox
    ```
    أضفه كملف تعريف بعيد.

4. اتصل وتحقق من تدفق الحركة — تحقق من استهلاك المستخدم في لوحة التحكم

---

## 6. تفعيل الإشعارات

أعدّ إشعارات تيليجرام للبقاء على اطلاع:

1. اذهب إلى **الإعدادات → الإشعارات → تيليجرام**
2. أدخل **توكن البوت** (أنشئ واحداً عبر [@BotFather](https://t.me/BotFather))
3. أدخل **معرّف محادثة المسؤول** (احصل عليه من [@userinfobot](https://t.me/userinfobot))
4. انقر **اختبار** للتحقق من التسليم
5. فعّل الأحداث المطلوبة:
    - إنشاء/انتهاء/تقييد مستخدم
    - انقطاع عقدة
    - وصول عتبة الحصة
    - اكتمال النسخ الاحتياطي
