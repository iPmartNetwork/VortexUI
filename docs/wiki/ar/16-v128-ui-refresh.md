## التنقل

### الشريط الجانبي

يجمع الصفحات حسب الأقسام (Overview، Users، Nodes، Network، Security، Reseller، …).
اضغط **السهم** أسفل الشريط للوضع **المصغّر** (أيقونات فقط) — مرّر على أيقونة لرؤية التسمية.

على الموبايل افتح القائمة من زر **الهامبرger** في الرأس.

### لوحة الأوامر

**Ctrl+K** (Windows/Linux) أو **⌘K** (macOS)، أو انقر حقل البحث في الرأس. اكتب اسم
الصفحة واضغط Enter.

### مبدّل اللغة

أيقونة **globe** في الرأس أو صفحة الدخول. الثماني لغات تشترك في نفس مجموعة المفاتيح.

### اختصار بوابة المستخدم

**User Portal** في الشريط أو الرأس → `/portal/login` للخدمة الذاتية (الاستهلاك،
الخطط، تذاكر الدعم).

## للمطورين

ملفات الترجمة في `web/src/i18n/locale/*.json`. بعد التعديل:

```bash
cd web
node scripts/apply-i18n-locales.mjs
node scripts/check-i18n.mjs
npm run build
```
