# لوحة المعلومات

!!! info "الوقت الفعلي"
    جميع بيانات لوحة المعلومات تُحدَّث عبر أحداث مرسلة من الخادم (SSE) — لا حاجة للتحديث اليدوي.

---

## الأدوات المصغرة (Widgets)

لوحة المعلومات الرئيسية عبارة عن **شبكة أدوات قابلة للتخصيص**. اسحب وأفلت وغيّر حجم الأدوات لبناء التخطيط المفضل لديك.

### الأدوات المتاحة

| الأداة | المحتوى |
|--------|---------|
| ملخص المستخدمين | الإجمالي، النشطون، المقيّدون، المنتهون |
| ملخص الحركة | رفع/تنزيل اليوم، الإجمالي الشهري |
| حالة العقد | عدد العقد المتصلة/غير المتصلة مع مؤشرات السلامة |
| مقاييس النظام | وحدة المعالجة، الذاكرة، القرص (حلقات متحركة) |
| الاتصالات النشطة | عدد الأنفاق الحية الحالية |
| الأحداث الأخيرة | أحدث أحداث النظام (إنشاء مستخدم، انقطاع عقدة، إلخ) |
| إجراءات سريعة | اختصارات لإنشاء مستخدم، إضافة عقدة، تشغيل فحص |
| حصة الموزّع | الحصة المتبقية من الحسابات/الحركة (عرض الموزّع) |

### التخصيص

1. انقر **أيقونة الشبكة** (أعلى يمين لوحة المعلومات)
2. اسحب الأدوات لإعادة ترتيبها
3. غيّر الحجم بسحب مقبض الزاوية
4. بدّل رؤية الأدوات بمربعات الاختيار
5. انقر **حفظ التخطيط** — يُحفظ لكل حساب مسؤول

---

## المقاييس الحيّة

مقاييس متحركة تعرض إحصائيات النظام الحيّة لكل عقدة:

- **المعالج** — نسبة الاستخدام الحالية
- **الذاكرة** — المستخدمة / الإجمالية مع النسبة
- **النطاق** — الإنتاجية الحالية (رفع + تنزيل)
- **الاتصالات** — عدد الأنفاق النشطة

تُحدَّث المقاييس كل 3 ثوانٍ عبر دفع SSE.

---

## الرسوم البيانية

| الرسم | الوصف |
|-------|-------|
| الحركة (خطي) | رفع/تنزيل عبر الزمن (ساعي/يومي/أسبوعي) |
| الاتصالات (مساحي) | الاتصالات النشطة عبر الزمن |
| المستخدمون (أعمدة) | المستخدمون الجدد حسب اليوم/الأسبوع |
| حمل العقد (مكدّس) | توزيع وحدة المعالجة لكل عقدة |

تدعم جميع الرسوم اختيار النطاق الزمني: **24 ساعة**، **7 أيام**، **30 يوماً**، أو نطاق تواريخ مخصص.

---

## التصوّر الجغرافي (خريطة العالم)

خريطة حرارية تُظهر من أين يتصل مستخدموك، بناءً على بيانات GeoIP من عملاء العقد.

- مرّر فوق دولة لعرض عدد الاتصالات وحجم الحركة
- انقر للحصول على تفاصيل (أعلى المدن، أعلى المستخدمين من تلك الدولة)
- تعكس كثافة اللون حجم الحركة

!!! note
    تتطلب البيانات الجغرافية أن يُبلّغ عملاء العقد عن بيانات الاتصال. إذا كانت الخريطة فارغة، تحقق أن عقدك تشغّل أحدث إصدار من العميل.

## صفحة المراقبة

**لوحة المعلومات → المراقبة** — عرض فوري لجميع الاتصالات النشطة.

| العمود | الوصف |
|--------|-------|
| المستخدم | اسم المستخدم |
| العقدة | الخادم المتصل به |
| IP | عنوان IP المصدر للعميل |
| البروتوكول | VLESS، VMess، Trojan، إلخ |
| المدة | مدة الاتصال النشط |
| الحركة | رفع/تنزيل لهذه الجلسة |

المميزات:

- تحديث تلقائي كل 3 ثوانٍ
- تصفية حسب العقدة، البروتوكول، أو المستخدم
- ترتيب حسب أي عمود
- انقر على مستخدم للانتقال إلى صفحة تفاصيله

---

## صفحة التحليلات

**لوحة المعلومات → التحليلات** — رؤى الحركة مجمّعة حسب الدولة والمستخدم والوقت.

### الأقسام

| القسم | يعرض |
|-------|------|
| بطاقات الملخص | إجمالي الرفع، التنزيل، عدد الدول |
| الحركة حسب الدولة | تحليل جغرافي — الدولة، الاتصالات، البايتات |
| أعلى المستخدمين | مرتبون حسب إجمالي الحركة المستهلكة |
| ساعات الذروة | رسم أعمدة لحجم الحركة بالساعة |

### النطاقات الزمنية

اختر **آخر 24 ساعة**، **آخر 7 أيام**، **آخر 30 يوماً**، أو نطاق تواريخ مخصص.

### التصدير

انقر **تصدير CSV** لتنزيل البيانات الجغرافية والمستخدمين كجدول بيانات للنطاق الزمني المحدد.

!!! tip
    استخدم التحليلات لتحديد ساعاتك الأكثر ازدحاماً والتخطيط لسعة العقد وفقاً لذلك.

## لوحة الأوامر

اضغط ++ctrl+k++ (أو ++cmd+k++ على macOS) لفتح لوحة الأوامر — بحث ضبابي عبر لوحة التحكم بأكملها.

### ما يمكنك البحث عنه

- **المستخدمون** — الانتقال لأي مستخدم بالاسم
- **العقد** — الانتقال لتفاصيل العقدة
- **الاتصالات الواردة** — فتح إعدادات الاتصال الوارد
- **الصفحات** — الانتقال لأي صفحة في لوحة التحكم
- **الإجراءات** — إنشاء مستخدم، إضافة عقدة، تشغيل نسخ احتياطي، إلخ

### اختصارات لوحة المفاتيح

| الاختصار | الإجراء |
|----------|---------|
| ++ctrl+k++ | فتح لوحة الأوامر |
| ++ctrl+shift+n++ | مستخدم جديد |
| ++ctrl+shift+d++ | تبديل الوضع الداكن/الفاتح |
| ++ctrl+period++ | فتح الإشعارات |
| ++escape++ | إغلاق النافذة/لوحة الأوامر |
| ++g++ ثم ++d++ | الانتقال للوحة المعلومات |
| ++g++ ثم ++u++ | الانتقال للمستخدمين |
| ++g++ ثم ++n++ | الانتقال للعقد |
