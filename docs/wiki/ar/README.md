## الجديد في الإصدار 1.4.0

- **التبديل التلقائي للبروتوكول** — تجاوز تلقائي ذاتي الإصلاح
- **محرك التكوين الذكي** — تحسين anti-DPI حسب مزود الخدمة
- **تدوير SNI الديناميكي** — تدوير يومي من مجموعات النطاقات
- **توجيه Multi-CDN** — IP نظيف CF/Arvan/Gcore
- **Mux ذكي** — تعدد إرسال محسّن لكل ISP
- **نقاط الجودة** — تسجيل وإعادة ترتيب تلقائي
- **منع تسرب DNS** — DoH + حظر DNS العادي
- **الرجوع الطارئ** — مخرج الملاذ الأخير
