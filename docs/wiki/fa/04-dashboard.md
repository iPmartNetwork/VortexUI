## ویجت‌های نمای کلی

داشبورد اصلی یک **گرید ویجت قابل سفارشی‌سازی** است. ویجت‌ها را درگ، دراپ و تغییر اندازه دهید تا چیدمان دلخواه‌تان ساخته شود.

### ویجت‌های موجود

| ویجت | محتوا |
|------|-------|
| خلاصه کاربران | تعداد کل، فعال، محدودشده، منقضی‌شده |
| خلاصه ترافیک | آپلود/دانلود امروز، مجموع ماهانه |
| وضعیت نودها | تعداد نودهای آنلاین/آفلاین با نشانگر سلامت |
| نمودارهای سیستم | مصرف CPU، RAM، دیسک (حلقه‌های متحرک) |
| اتصالات فعال | تعداد تانل‌های زنده فعلی |
| رویدادهای اخیر | آخرین رویدادهای سیستم (کاربر ایجاد شد، نود آفلاین شد و ...) |
| اقدامات سریع | میانبرهای ایجاد کاربر، افزودن نود، اجرای اسکن |
| استخر ریسلر | سهمیه اکانت/ترافیک باقیمانده (نمای ریسلر) |

### سفارشی‌سازی

1. روی **آیکون گرید** (بالا-راست داشبورد) کلیک کنید
2. ویجت‌ها را برای تغییر ترتیب درگ کنید
3. با کشیدن گوشه‌ها تغییر اندازه دهید
4. نمایش ویجت‌ها را با چک‌باکس تنظیم کنید
5. روی **ذخیره چیدمان** کلیک کنید — به ازای هر اکانت ادمین ذخیره می‌شود

## نمودارهای بلادرنگ

نمودارهای متحرک، متریک‌های زنده سیستم را برای هر نود نشان می‌دهند:

- **CPU** — درصد فعلی استفاده
- **RAM** — مصرف‌شده / کل با درصد
- **پهنای‌باند** — توان عبور فعلی (آپلود + دانلود)
- **اتصالات** — تعداد تانل‌های فعال

نمودارها هر ۳ ثانیه از طریق SSE بروزرسانی می‌شوند.

---

## چارت‌ها

| چارت | توضیحات |
|------|---------|
| ترافیک (خطی) | آپلود/دانلود در طول زمان (ساعتی/روزانه/هفتگی) |
| اتصالات (مساحتی) | اتصالات فعال در طول زمان |
| کاربران (میله‌ای) | کاربران جدید به تفکیک روز/هفته |
| بار نودها (انباشته) | توزیع CPU به ازای هر نود |

همه چارت‌ها از انتخاب بازه زمانی پشتیبانی می‌کنند: **۲۴ ساعت**، **۷ روز**، **۳۰ روز** یا بازه سفارشی.

## نقشه جغرافیایی جهان

نقشه حرارتی نشان می‌دهد کاربران از کجا متصل می‌شوند، بر اساس داده GeoIP از ایجنت‌های نود.

- ماوس را روی یک کشور نگه دارید تا تعداد اتصالات و حجم ترافیک نمایش داده شود
- کلیک کنید تا جزئیات (شهرهای برتر، کاربران برتر آن کشور) نشان داده شود
- شدت رنگ نشان‌دهنده حجم ترافیک است

!!! note
    داده جغرافیایی نیاز دارد ایجنت‌های نود متادیتای اتصال را گزارش دهند. اگر نقشه خالی است، بررسی کنید نودها آخرین نسخه ایجنت را اجرا می‌کنند.

## صفحه مانیتور

**داشبورد → مانیتور** — نمای بلادرنگ تمام اتصالات فعال.

| ستون | توضیحات |
|------|---------|
| کاربر | نام کاربری |
| نود | به کدام سرور متصل است |
| IP | آی‌پی مبدأ کلاینت |
| پروتکل | VLESS، VMess، Trojan و غیره |
| مدت | چه مدت اتصال فعال بوده |
| ترافیک | آپلود/دانلود این نشست |

ویژگی‌ها:

- هر ۳ ثانیه به‌صورت خودکار رفرش می‌شود
- فیلتر بر اساس نود، پروتکل یا کاربر
- مرتب‌سازی بر اساس هر ستون
- کلیک روی کاربر برای رفتن به صفحه جزئیات

## صفحه تحلیل‌ها

**داشبورد → تحلیل‌ها** — بینش‌های ترافیکی تجمیع‌شده بر اساس کشور، کاربر و زمان.

### بخش‌ها

| بخش | نمایش |
|-----|-------|
| کارت‌های خلاصه | مجموع آپلود، دانلود، تعداد کشورها |
| ترافیک بر اساس کشور | تفکیک جغرافیایی — کشور، اتصالات، بایت |
| کاربران برتر | رتبه‌بندی بر اساس کل ترافیک مصرف‌شده |
| ساعات اوج | چارت میله‌ای حجم ترافیک ساعتی |

### بازه‌های زمانی

**۲۴ ساعت اخیر**، **۷ روز اخیر**، **۳۰ روز اخیر** یا بازه تاریخی سفارشی انتخاب کنید.

### خروجی

روی **خروجی CSV** کلیک کنید تا داده‌های جغرافیایی + کاربر به‌صورت اسپردشیت برای بازه انتخابی دانلود شود.

!!! tip
    از تحلیل‌ها برای شناسایی ساعات شلوغ و برنامه‌ریزی ظرفیت نودها استفاده کنید.

## پالت دستور

++ctrl+k++ (یا ++cmd+k++ در macOS) را فشار دهید تا پالت دستور باز شود — جستجوی فازی در تمام پنل.

### چه چیزهایی قابل جستجوست

- **کاربران** — پرش به هر کاربر با نام
- **نودها** — رفتن به جزئیات نود
- **اینباندها** — باز کردن تنظیمات اینباند
- **صفحات** — رفتن به هر صفحه پنل
- **اقدامات** — ایجاد کاربر، افزودن نود، اجرای بکاپ و غیره

### میانبرهای صفحه‌کلید

| میانبر | عملکرد |
|--------|--------|
| ++ctrl+k++ | باز کردن پالت دستور |
| ++ctrl+shift+n++ | کاربر جدید |
| ++ctrl+shift+d++ | تغییر تم تاریک/روشن |
| ++ctrl+period++ | باز کردن اعلان‌ها |
| ++escape++ | بستن مودال/پالت |
| ++g++ سپس ++d++ | رفتن به داشبورد |
| ++g++ سپس ++u++ | رفتن به کاربران |
| ++g++ سپس ++n++ | رفتن به نودها |
