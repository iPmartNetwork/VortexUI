## تازه‌های نسخه ۱.۴.۰

- **سوئیچ خودکار پروتکل** — فیلاور خودترمیم بین پروتکل‌ها
- **موتور کانفیگ هوشمند** — بهینه‌سازی anti-DPI بر اساس ISP
- **چرخش SNI** — تغییر روزانه از pool دامنه‌های ISP
- **مسیریابی Multi-CDN** — IP تمیز CF/Arvan/Gcore
- **Mux هوشمند** — مالتی‌پلکسینگ بهینه برای هر ISP
- **امتیاز کیفیت** — امتیازدهی و مرتب‌سازی خودکار proxy
- **جلوگیری نشت DNS** — DoH + بلاک DNS ساده
- **فال‌بک اضطراری** — خروجی آخرین راه‌حل
