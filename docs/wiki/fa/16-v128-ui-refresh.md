!!! info "فقط فرانت‌اند"
    نسخه ۱.۲.۸ یک **انتشار UI و i18n** است. migration دیتابیس یا تغییر API لازم نیست —

## تازه‌ها

| بخش | توضیح |
|-----|--------|
| **سیستم طراحی** | کارت شیشه‌ای، کاشی آمار، نشان وضعیت، پالت cyan/sky، انیمیشن ورود صفحه |
| **Shell** | سایدبار تاشو، هدر ثابت، حالت mini، کشوی موبایل |
| **Command palette** | **Ctrl+K** / **⌘K** برای رفتن سریع به هر صفحه ادمین |
| **صفحات اصلی** | Overview، Users و Nodes با کارت آمار زنده از API |
| **پورتال کاربری** | ورود، داشبورد، سایدبار دسکتاپ و ناوبری پایین موبایل |
| **i18n** | **۶۳۹ کلید** در EN / FA / TR / AR / RU / ZH / JA / ES |

## ناوبری

### سایدبار

صفحات در بخش‌های مختلف (Overview، Users، Nodes، Network، Security، Reseller و …)
گروه‌بندی شده‌اند. با **فلش پایین** سایدبار را به حالت **mini** (فقط آیکون) ببرید —
روی آیکون hover کنید تا عنوان را ببینید.

در موبایل منو را با **همبرگر** در هدر باز کنید.

### Command palette

**Ctrl+K** (ویندوز/لینوکس) یا **⌘K** (مک) بزنید، یا روی جستجو در هدر کلیک کنید.
نام صفحه را تایپ کنید و Enter بزنید.

### تغییر زبان

آیکون **globe** در هدر یا صفحه ورود. هر ۸ زبان مجموعه کلید یکسان دارند — billing،
پرداخت ریسلر، سفارش‌های در انتظار، shell و portal کامل ترجمه شده‌اند.

### میانبر پورتال

**پورتال کاربری** در سایدبار یا هدر → `/portal/login` برای مصرف، خرید پلن و تیکت.

## تم

دکمه **خورشید/ماه** در هدر — تم تیره/روشن در localStorage ذخیره می‌شود.

## توسعه‌دهندگان

فایل‌های تکمیلی: `web/src/i18n/locale/*.json`

```bash
cd web
node scripts/apply-i18n-locales.mjs
node scripts/check-i18n.mjs
npm run build
```
