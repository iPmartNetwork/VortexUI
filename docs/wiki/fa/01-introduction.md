# معرفی

## VortexUI چیست؟

**VortexUI** یک پنل مدیریت پروکسی نسل بعدی است که برای اپراتورهایی ساخته شده که نیاز دارند به:

- **مقیاس** — مدیریت هزاران کاربر در ده‌ها نود
- **تاب‌آوری** — failover خودکار، مانیتورینگ سلامت، ناوگان نود خودترمیم
- **ضد سانسور** — ترفندهای TLS مختص ISP، حفاظت probing، سایت‌های فریبنده، WARP+
- **سلف‌سرویس** — کاربران حساب خود را مدیریت کنند، پلن بخرند، تیکت باز کنند
- **درآمد** — پلن‌های ریسلر، درگاه‌های پرداخت متعدد، کیف پول، برنامه ارجاع
- **واگذاری** — پلتفرم کامل نمایندگی با زیرنمایندگی، وایت‌لیبل، محدودیت‌های سیاست

بر خلاف پنل‌های اینباند‌محور (3x-ui)، VortexUI از **مدل کاربرمحور** استفاده می‌کند: یک هویت کاربری دسترسی به تمام پروتکل‌های اختصاص‌یافته در تمام نودها را فراهم می‌کند.

> **نسخه فعلی: ۱.۳.۱** — اکنون با تنظیمات ماندگار، لاگ حسابرسی، گواهی‌های واقعی ACME، فدراسیون و پلتفرم کامل نمایندگی.

## مرور ویژگی‌ها

### موتور و زیرساخت

| قابلیت | جزئیات |
|--------|--------|
| پشتیبانی دو هسته | Xray-core و sing-box — انتخاب به ازای هر نود |
| ترافیک دلتا | مقاوم در برابر ری‌استارت، بدون شمارش دوباره |
| ناوگان نود mTLS | اتصالات رمزنگاری‌شده، failover خودکار |
| ویزارد ثبت‌نام نود | رابط ۴ مرحله‌ای برای آنبوردینگ نودهای ریموت |
| مهاجرت خودکار | انتقال کاربران از نودهای ناسالم |
| فدراسیون | همگام‌سازی کاربران/نودها بین چند پنل |
| نود محلی | هسته در پروسه — بدون نیاز به agent جداگانه |
| زنجیره CDN/Relay | مسیرهای چندپرش با CDN، ریلی و ورکر |
| لودبالانسر | ۴ استراتژی با بررسی سلامت |
| اتوماسیون DNS کلودفلر | مدیریت خودکار رکوردهای DNS برای نودها |

### امنیت و ضد سانسور

| قابلیت | جزئیات |
|--------|--------|
| اسکنر Reality | کشف SNI‌های بهینه با امتیازدهی تأخیر |
| مدیر ترفندهای TLS | پروفایل‌های ISP (fragment، mux، padding) |
| حفاظت Probing | تشخیص و مسدودسازی probe‌های GFW |
| اعتبارسنجی اثرانگشت | فیلتر کلاینت بر اساس JA3 |
| وبسایت فریبنده | نمایش سایت جعلی به سانسورچی‌ها |
| DNS-over-HTTPS | DoH داخلی با مسدودسازی تبلیغات/بدافزار |
| پروفایل‌های فرار | پیش‌تنظیمات ضد DPI به ازای هر کشور |
| یکپارچگی WARP+ | خروجی کلودفلر برای IP تمیز |
| اسکنر Clean-IP | یافتن بهترین IP‌های CDN با امتیازدهی |
| اجرای محدودیت IP | سقف IP همزمان با اقدامات قابل تنظیم |
| مسدودسازی جغرافیایی | محدودیت کشور به ازای هر اینباند |
| محافظ اشتراک‌گذاری | تشخیص و اقدام روی اشتراک حساب |

### مدیریت کاربر و تجارت

| قابلیت | جزئیات |
|--------|--------|
| پورتال سلف‌سرویس | ورود با توکن، مشاهده مصرف، تیکت |
| فروشگاه سلف‌سرویس | پلن‌های ریسلر با کارت/کریپتو/زرین‌پال |
| سهمیه هوشمند | کاهش تدریجی سرعت |
| گروه‌های خانوادگی | مخزن داده مشترک برای چند کاربر |
| سیستم ارجاع | کدهای دعوت با پاداش ترافیک/روز |
| پلن‌های ریسلر | هر ریسلر پلن و قیمت خود را ایجاد کند |
| درگاه‌های پرداخت | زرین‌پال، کارت‌به‌کارت، کریپتو |
| کیف پول | سیستم اعتبار ریسلر با صف شارژ |
| هاست‌های اشتراک | override آدرس/CDN با متغیرهای قالب |
| دیپ لینک + QR | وارد کردن یک‌ضربه‌ای اشتراک |
| قالب‌های کانفیگ | روتینگ سفارشی Clash/sing-box |
| ابزارهای ایمپورت | مهاجرت از 3x-ui یا Marzban |

### مدیریت و پلتفرم نمایندگی

| قابلیت | جزئیات |
|--------|--------|
| RBAC + نقش‌ها | مجوزهای دقیق به ازای هر ادمین |
| پلتفرم کامل نمایندگی | کیف پول، زیرنمایندگی، وایت‌لیبل، وبهوک |
| لیست‌های مجاز محدوده‌دار | محدودیت پلن/نود/اینباند به ازای ریسلر |
| تعلیق خودکار | تعلیق خودکار ریسلر در تخلفات |
| لاگ حسابرسی | ردیابی هر اقدام ادمین با diff |
| اعلان‌های سهمیه | آستانه‌های قابل تنظیم برای هشدار ریسلر |
| پشتیبان‌گیری خودکار | صادرات زمان‌بندی‌شده به تلگرام یا S3 |
| متریک‌های Grafana | اندپوینت پرومتئوس + داشبورد آماده |

## معماری

```
┌──────────────────────────────────────────────────────────────┐
│  Caddy (لایه وب)       — HTTPS, SPA, reverse proxy, DoH     │
├──────────────────────────────────────────────────────────────┤
│  Panel (Go 1.26)       — REST API, SSE, gRPC hub, scheduler │
│  ├─ Auth               — JWT + TOTP + توکن‌های پورتال        │
│  ├─ Services           — کاربر، نود، پلن، سفارش، تحلیلات    │
│  ├─ Hub                — مدیریت ناوگان نود + failover        │
│  ├─ Scanner            — Reality SNI prober + Clean-IP       │
│  ├─ Migration          — توزیع مجدد کاربر بر اساس سلامت     │
│  ├─ Reseller           — کیف پول، پلن‌ها، برندینگ، وبهوک    │
│  └─ Federation         — همگام‌سازی بین‌پنلی                  │
├──────────────────────────────────────────────────────────────┤
│  PostgreSQL + TimescaleDB — داده + ترافیک سری‌زمانی          │
│  Redis                    — کش، سشن، ردیاب دستگاه           │
├──────────────────────────────────────────────────────────────┤
│  Node Agent (gRPC)     — اجرای هسته ریموت + سلامت           │
│  Local Node            — در پروسه روی هاست پنل              │
└──────────────────────────────────────────────────────────────┘
```
