# شروع سریع

!!! info "زمان تخمینی"
    از ورود تا یک لینک سابسکریپشن کارا: **۵ دقیقه**.

---

## ۱. ورود و ایمن‌سازی اکانت

1. به URL پنل خود بروید (مثلاً `https://panel.example.com`)
2. با اعتبارنامه ادمینی که هنگام نصب ایجاد کردید وارد شوید
3. **فوراً رمز عبور را تغییر دهید**: تنظیمات → پروفایل → تغییر رمز عبور

### فعال‌سازی احراز هویت دو مرحله‌ای (TOTP)

1. به **تنظیمات → احراز هویت دو مرحله‌ای** بروید
2. روی **فعال‌سازی** کلیک کنید — کد QR را با Google Authenticator یا هر اپ TOTP اسکن کنید
3. کد ۶ رقمی را برای تأیید وارد کنید
4. کدهای بازیابی را در جای امنی ذخیره کنید

!!! warning
    بدون فعال‌سازی 2FA، هرکسی که رمز عبور شما را داشته باشد دسترسی کامل به پنل خواهد داشت. قبل از اکسپوز کردن پنل به اینترنت آن را فعال کنید.

---

## ۲. افزودن اولین نود

=== "نود محلی (تک سرور)"

    اگر پنل و هسته پروکسی روی یک ماشین اجرا می‌شوند:

    1. به **نودها → افزودن نود** بروید
    2. **نود محلی** را انتخاب کنید
    3. هسته را انتخاب کنید: **Xray-core** یا **sing-box**
    4. روی **ایجاد** کلیک کنید
    5. نود به‌صورت خودکار شروع به کار کرده و "آنلاین" نمایش داده می‌شود

=== "نود ریموت (ویزارد ثبت‌نام)"

    برای یک سرور جداگانه:

    1. به **نودها → افزودن نود** بروید
    2. **نود ریموت** را انتخاب کنید
    3. ویزارد یک دستور نصب تک‌خطی نمایش می‌دهد — آن را کپی کنید
    4. به سرور ریموت SSH کرده و دستور را paste کنید
    5. منتظر اتصال ایجنت باشید — نود با وضعیت "آنلاین" نمایش داده می‌شود

!!! tip
    ویزارد ثبت‌نام تبادل گواهی، نصب هسته و ثبت سرویس را مدیریت می‌کند. نیازی به مدیریت دستی گواهی نیست.

---

## ۳. ایجاد اولین اینباند

یک راه‌اندازی سریع VLESS + REALITY (پیشنهادی برای اتصالات مقاوم در برابر سانسور):

1. به **اینباندها → افزودن اینباند** بروید
2. تنظیم کنید:

    | فیلد | مقدار |
    |------|-------|
    | پروتکل | VLESS |
    | نود | نود شما |
    | پورت | `443` |
    | ترنسپورت | TCP |
    | امنیت | REALITY |
    | مقصد (SNI) | `www.google.com:443` (یا از اسکنر Reality استفاده کنید) |
    | نام‌های سرور | `www.google.com` |
    | شناسه‌های کوتاه | تولید خودکار (یا خودتان وارد کنید) |

3. روی **ایجاد** کلیک کنید

!!! tip "اسکنر Reality"
    از **امنیت → اسکنر Reality** برای یافتن بهترین دامنه‌های SNI برای موقعیت سرورتان استفاده کنید. نتیجه با بالاترین امتیاز را برای تأخیر کم و اتصالات پایدار انتخاب کنید.

---

## ۴. ایجاد اولین کاربر

1. به **کاربران → کاربر جدید** بروید
2. فیلدها را پر کنید:

    | فیلد | مقدار |
    |------|-------|
    | نام کاربری | `testuser` |
    | حد ترافیک | `50 GB` (یا `0` برای نامحدود) |
    | انقضا | ۳۰ روز از الان |
    | حد دستگاه | `3` |
    | اینباندها | اینباند VLESS خود را انتخاب کنید |

3. روی **ایجاد** کلیک کنید
4. توکن سابسکریپشن کاربر به‌صورت خودکار تولید می‌شود

---

## ۵. تست لینک سابسکریپشن

1. از لیست کاربران، روی نام کاربر کلیک کنید تا صفحه جزئیات باز شود
2. **لینک سابسکریپشن** را کپی کنید (یا کد QR را اسکن کنید)
3. در اپ کلاینت وارد کنید:

=== "v2rayNG (اندروید)"

    1. v2rayNG را باز کنید → روی **+** ضربه بزنید → **وارد کردن کانفیگ از کلیپبورد**
    2. یا مستقیماً کد QR را اسکن کنید

=== "Clash Meta (دسکتاپ/موبایل)"

    ```
    https://panel.example.com/sub/<token>?format=clash
    ```
    به‌عنوان URL سابسکریپشن در Clash اضافه کنید.

=== "sing-box (iOS/Android)"

    ```
    https://panel.example.com/sub/<token>?format=singbox
    ```
    به‌عنوان پروفایل ریموت اضافه کنید.

4. متصل شوید و جریان ترافیک را تأیید کنید — مصرف کاربر را در پنل بررسی کنید

---

## ۶. فعال‌سازی اعلان‌ها

اعلان‌های تلگرام را برای اطلاع از رویدادها تنظیم کنید:

1. به **تنظیمات → اعلان‌ها → تلگرام** بروید
2. **توکن بات** را وارد کنید (از [@BotFather](https://t.me/BotFather) بسازید)
3. **شناسه چت ادمین** را وارد کنید (از [@userinfobot](https://t.me/userinfobot) دریافت کنید)
4. روی **تست** کلیک کنید تا ارسال تأیید شود
5. رویدادهای دلخواه را فعال کنید:
    - کاربر ایجاد/منقضی/محدود شد
    - نود آفلاین شد
    - آستانه سهمیه رسید
    - بکاپ تکمیل شد
