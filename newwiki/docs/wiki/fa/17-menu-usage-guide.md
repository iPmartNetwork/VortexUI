# راهنمای منوها و استفاده

این صفحه توضیح می‌دهد هر منوی اصلی VortexUI چه کاری انجام می‌دهد و بعد از نصب چطور باید از پنل استفاده کنید.

---

## نمای کلی منوها

| منو | کاربرد | مناسب برای |
|-----|--------|------------|
| Overview / Dashboard | وضعیت زنده سیستم، نودها، ترافیک و ویجت‌ها | ادمین اصلی، ادمین، ریسلر |
| Monitor | نمایش اتصال‌های فعال به صورت زنده | ادمین، پشتیبانی |
| Analytics | تحلیل مصرف بر اساس کاربر، کشور، زمان و پروتکل | ادمین، ریسلر |
| Users | ساخت، ویرایش، تمدید، تعلیق و خروجی کاربران | ادمین، ریسلر |
| Families | ساخت گروه با ترافیک اشتراکی | ادمین، ریسلر |
| Nodes | افزودن و مانیتور کردن سرورهای Xray یا sing-box | ادمین |
| Inbounds | ساخت پروتکل‌ها، پورت‌ها و تنظیمات ورودی | ادمین |
| Routing | روتینگ، خروجی‌ها، WARP، CDN chain و لودبالانسر | ادمین |
| Evasion / Security Suite | Reality، Clean-IP، TLS Tricks، Decoy و ضد probing | ادمین |
| Plans & Payments | پلن‌ها، سفارش‌ها، روش‌های پرداخت و فاکتور | ادمین، ریسلر |
| Wallet Billing | کیف پول ریسلر، شارژ، دفترکل و سهمیه‌ها | ادمین اصلی، ریسلر |
| Tickets | تیکت‌های پشتیبانی کاربران | ادمین، ریسلر |
| Audit | لاگ کامل اقدامات ادمین‌ها | ادمین اصلی، Auditor |
| Settings | تنظیمات، ادمین‌ها، بکاپ، ACME، API token | ادمین اصلی |

## گردش‌کار کامل بعد از نصب

### 1. نصب

یکی از دو روش را انتخاب کنید:

- Docker: به صفحه [نصب](02-installation.md) مراجعه کنید.
- Native: به بخش بیلد بومی در [نصب](02-installation.md) مراجعه کنید.

### 2. امن‌سازی پنل

1. با ادمین اصلی وارد شوید.
2. رمزعبور را تغییر دهید.
3. 2FA را فعال کنید.
4. IP Guard یا whitelist را تنظیم کنید.
5. بکاپ خودکار را فعال کنید.
6. در صورت نیاز ACME را برای گواهی Let's Encrypt تنظیم کنید.

### 3. افزودن نود

1. به **Nodes → Add Node** بروید.
2. نود Local یا Remote را انتخاب کنید.
3. برای نود Remote دستور نصب تولیدشده را روی سرور اجرا کنید.
4. وضعیت نود باید Online شود.

### 4. ساخت Inbound

برای اکثر شرایط، پیشنهاد اصلی:

| سناریو | پیشنهاد |
|--------|---------|
| مقاومت در برابر سانسور | VLESS + TCP + REALITY |
| پشت CDN | VLESS/VMess + WebSocket + TLS |
| سرعت بالا | Trojan + gRPC + TLS |
| شبکه‌های ناپایدار | Hysteria2 یا TUIC |

مراحل:

1. به **Inbounds → Add Inbound** بروید.
2. نود را انتخاب کنید.
3. پروتکل، پورت، ترانسپورت و امنیت را تنظیم کنید.
4. برای REALITY از Reality Scanner کمک بگیرید.
5. ذخیره کنید.

### 5. ساخت کاربر

1. به **Users → Add User** بروید.
2. نام کاربری، حجم، تاریخ انقضا و محدودیت دستگاه را وارد کنید.
3. Inboundهای مجاز را انتخاب کنید.
4. کاربر را ذخیره کنید.
5. لینک subscription یا QR را برای کاربر ارسال کنید.

### 6. فعال‌سازی فروشگاه

1. در **Plans & Payments → Plans** پلن بسازید.
2. روش پرداخت مثل زرین‌پال، کارت‌به‌کارت یا کریپتو را فعال کنید.
3. کاربران از مسیر `/sub/{token}/shop` می‌توانند خرید یا تمدید کنند.

### 7. راه‌اندازی ریسلر

1. در **Settings → Admins** یک ادمین با نقش Reseller بسازید.
2. سهمیه کاربر و ترافیک تعیین کنید.
3. اعتبار کیف پول بدهید.
4. اجازه ساخت پلن و روش پرداخت را تنظیم کنید.
5. در صورت نیاز وایت‌لیبل پورتال را فعال کنید.

## استفاده روزانه

### برای ادمین اصلی

- وضعیت نودها را در Dashboard بررسی کنید.
- Audit Log را مرور کنید.
- سفارش‌ها و شارژهای ریسلر را بررسی کنید.
- بکاپ و سلامت ACME را کنترل کنید.

### برای ریسلر

- کاربران خودش را ایجاد و تمدید کند.
- پلن و روش پرداخت خودش را مدیریت کند.
- موجودی کیف پول و دفترکل را بررسی کند.
- تیکت‌های کاربران خود را پاسخ دهد.

### برای کاربر نهایی

- وارد پورتال شود.
- مصرف را ببیند.
- کانفیگ یا QR را دریافت کند.
- پلن بخرد یا تمدید کند.
- تیکت پشتیبانی ثبت کند.

## نکات پیشنهادی

- برای شروع سریع از VLESS + REALITY استفاده کنید.
- برای امنیت بیشتر Decoy و Probing Protection را فعال کنید.
- برای پروداکشن حتماً بکاپ خودکار فعال باشد.
- برای چند سرور، از نود Remote و Load Balancer استفاده کنید.
- برای فروش، Reseller Platform و Shop را فعال کنید.
